# Section 1: Introduction & Setup

## Overview

This section introduces GitHub Copilot CLI and ensures everyone has a working environment. You'll understand what Copilot CLI is, how it differs from other Copilot experiences, and complete your first interaction.

## Learning Objectives

By the end of this section, you will be able to:
- Explain what GitHub Copilot CLI is and its role in AI-driven development
- Identify key differences between Copilot CLI and IDE Copilot
- Navigate the basic interface and understand the security model
- Successfully complete your first Copilot CLI interaction

## Duration

**15 minutes**

## Key Takeaways

- Copilot CLI is a terminal-native AI assistant with full shell and GitHub access
- Security is built-in: explicit approval required for file modifications and commands
- Works with your existing Git workflow seamlessly
