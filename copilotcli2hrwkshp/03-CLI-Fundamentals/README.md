# Section 3: CLI Fundamentals

## Overview

This section covers the essential commands, keyboard shortcuts, and interaction patterns for working efficiently with GitHub Copilot CLI.

## Learning Objectives

By the end of this section, you will be able to:
- Use all essential Copilot CLI commands fluently
- Navigate between interaction modes effectively
- Apply prompt techniques for better results
- Establish efficient terminal-first workflows

## Duration

**15 minutes**

## Prerequisites

- Completed Sections 1-2
- Working Copilot CLI installation

## Key Takeaways

- Master the essential commands: `/help`, `/clear`, `/diff`, `/review`, `/mcp`
- Use `Shift+Tab` to switch modes: Interactive ↔ Plan ↔ Auto
- Reference files with `@`, run commands with `!`, reference issues with `#`
- Be specific in prompts — clear context leads to better results
