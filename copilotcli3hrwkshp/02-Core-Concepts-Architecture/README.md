# Section 2: Core Concepts & Architecture

## Overview

This section dives deep into how GitHub Copilot CLI works internally, how it builds context, and the configuration system that powers its flexibility. Understanding these concepts will help you use Copilot CLI more effectively.

## Learning Objectives

By the end of this section, you will be able to:
- Explain the Copilot CLI architecture and interaction model
- Describe the configuration hierarchy (local, user, repository)
- Identify special files and their purposes
- Understand how Copilot builds and utilizes context
- Navigate the configuration directory structure
