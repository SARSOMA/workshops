# Section 6: Extensibility with MCP (Model Context Protocol)

## Overview

MCP (Model Context Protocol) allows Copilot CLI to connect with external systems, tools, and data sources. This section covers configuring MCP servers to enhance Copilot's capabilities.

## Learning Objectives

By the end of this section, you will be able to:
- Explain what MCP is and why it matters
- Configure MCP servers in Copilot CLI
- Connect to common services (GitHub, Azure DevOps, etc.)
- Understand how to extend Copilot with real-time data
