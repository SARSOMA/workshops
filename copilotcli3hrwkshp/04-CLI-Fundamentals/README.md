# Section 4: Copilot CLI Fundamentals

## Overview

This section covers the core CLI experience—the commands, workflows, and interaction patterns that make Copilot CLI powerful for terminal-first development.

## Learning Objectives

By the end of this section, you will be able to:
- Navigate the CLI interface fluently
- Use all essential slash commands effectively
- Master keyboard shortcuts for efficiency
- Work with modes for different task types
- Apply terminal-first development workflows

- Slash commands and their categories
