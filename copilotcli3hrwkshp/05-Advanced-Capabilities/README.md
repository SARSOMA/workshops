# Section 5: Advanced Capabilities

## Overview

This section covers advanced features that make Copilot CLI a powerful automation tool—custom agents, skills, programmatic usage, and hooks.

## Learning Objectives

By the end of this section, you will be able to:
- Create and use custom agents for specialized tasks
- Define and use skills for repeatable automation
- Use Copilot CLI programmatically in scripts
- Configure hooks for workflow automation
- Apply advanced command-line options

## Tutorials in This Section

| Tutorial | Duration | Description |
|----------|----------|-------------|

| [5.2 Skills & Automation](./02-skills-automation.md) | 15 min | Building reusable skills |

## Key Concepts

- Agent profiles and customization
- Skill definitions and resources
- Command-line flags and options
- Hooks for lifecycle events
- Sub-agent delegation

## Prerequisites

- Completed Sections 1-4
- Familiarity with Markdown
- Basic scripting knowledge (helpful)
